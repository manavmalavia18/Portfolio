# Manav Malavia — Portfolio

Personal portfolio site for [manavmalavia.org](https://manavmalavia.org).

Built with Next.js, TypeScript, Tailwind CSS, and Framer Motion. Based on the [kintarowwwards](https://github.com/xkintaro/kintarowwwards) template.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deploy on [Vercel](https://vercel.com) and connect the `manavmalavia.org` domain.
